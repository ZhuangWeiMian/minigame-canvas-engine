# Image

继承自 [Element](/components/element.html)。

## 特殊属性
|      属性      |  类型  | 是否必填 |          说明          |
|----------------|--------|--------|------------------------|
| src| string |    是 |图片链接, 修改图片的src属性会自动请求新的图片并渲染|

::: tip
1.小游戏开放数据域场景图片路径不需要加./作为前缀，以小游戏根目录作为根目录；

2.图片可以通过API[loadImgs](/api/api.html#loadimgs)实现预加载
:::
